# ClientServerApplication

This project implements a simple client-server application using Python socket programming. The server listens for incoming connections on localhost, while the client connects to the server and exchanges messages. The application demonstrates the client-server model, layered architecture, and successful communication between distributed system components.

## Features
- TCP socket communication
- Localhost (127.0.0.1) implementation
- Message exchange between client and server
- Simple and lightweight design
- Demonstrates client-server architecture concepts

## Files
- `server.py` - Server program that listens for connections and responds to client messages.
- `client.py` - Client program that connects to the server and sends messages.

## Requirements
- Python 3.x
- Visual Studio Code (optional)
